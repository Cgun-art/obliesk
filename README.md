# OBLIESK

an xbox distro to make potential.

OPEN SOURCE / CMAKE IGNORING

# How to setup obliesk

1: download the repository 

2: install python

3: run credits() to check python owners

4: close python

5: plug in a USB-A or an USB-C drive/pendrive

6: transfer obliesk data to the pendrive/drive

7: unplug the usb

8: get it back to your xbox one or your xbox 360

9: check for obliesk

10: done!
